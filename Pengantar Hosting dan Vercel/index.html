Pengenalan Hosting Web & Vercel

Apa itu Hosting Web?

Hosting web adalah layanan untuk menyimpan dan menampilkan website di internet agar bisa diakses oleh siapa saja. File website (HTML, CSS, JS, gambar, dll) ditempatkan di server yang selalu online dan terhubung ke internet.

Apa itu Vercel?

Vercel adalah platform hosting modern yang memudahkan proses deploy website. Dengan Vercel, Anda bisa meng-upload (deploy) website secara otomatis dari repository (misal GitHub), dan website langsung mendapatkan URL publik yang bisa diakses semua orang.

Cara Deploy Website ke Vercel

  1. Buat akun di vercel.com
  2. Hubungkan repository project Anda (misal dari GitHub)
  3. Klik "Deploy" dan tunggu proses selesai
  4. Website Anda langsung online dan mendapatkan URL publik

Vercel sangat cocok untuk web statis, landing page, portfolio, dan aplikasi modern. Setiap kali Anda mengubah kode dan push ke repository, website akan otomatis ter-update di Vercel.
