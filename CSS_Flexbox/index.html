Contoh Tata Letak Flexbox

Flexbox adalah metode modern untuk mengatur tata letak elemen secara fleksibel dan responsif. Dengan Flexbox, kita dapat dengan mudah mengatur posisi, arah, dan jarak antar elemen di dalam container.

  • display: flex — Mengaktifkan flexbox pada container.
  • flex-direction — Menentukan arah item (row/baris, column/kolom, row-reverse, column-reverse).
  • justify-content — Mengatur posisi item secara horizontal (start, center, end, space-between, space-around, space-evenly).
  • align-items — Mengatur posisi item secara vertikal (stretch, center, start, end, baseline).
  • gap — Mengatur jarak antar item.

Contoh di atas:
- Container menggunakan display: flex.
- justify-content: center membuat item berada di tengah secara horizontal.
- align-items: center membuat item berada di tengah secara vertikal.
- gap: 24px memberi jarak antar item.
- flex-direction: row (default) membuat item tersusun secara baris.
