Pengenalan Struktur Dokumen HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. Berikut adalah struktur dasar dokumen HTML:

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    ...konten di sini...
  </body>
</html>

Tag Umum dalam HTML

<p> : Paragraf
<a> : Tautan/Link
<img> : Gambar
