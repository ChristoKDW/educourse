Praktik Deploy Website ke Vercel

Langkah-langkah Deploy dari GitHub ke Vercel

  1. Buat akun di vercel.com
  2. Push project website Anda ke repository GitHub
  3. Di dashboard Vercel, klik "Add New Project" dan pilih repository GitHub Anda
  4. Ikuti instruksi, klik "Deploy" dan tunggu proses selesai
  5. Website Anda akan mendapatkan URL preview otomatis dari Vercel

Konfigurasi Domain di Vercel

  • Buka dashboard project di Vercel
  • Pilih menu "Domains"
  • Tambahkan custom domain (misal: namadomain.com)
  • Ikuti instruksi untuk mengarahkan DNS domain ke Vercel
  • Domain akan terhubung ke website Anda setelah propagasi DNS selesai

Preview Project

Setiap kali Anda melakukan perubahan dan push ke GitHub, Vercel akan otomatis membuat preview URL untuk setiap branch atau pull request. Anda bisa melihat hasil update sebelum website utama di-update.

Keuntungan Vercel: Proses deploy sangat mudah, update otomatis, dan mendukung custom domain serta preview project.
