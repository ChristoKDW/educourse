Perencanaan Final Project

1. Ide & Tujuan

Contoh ide: Aplikasi katalog kursus online sederhana untuk menampilkan materi, kuis singkat, dan tugas mini. Tujuan: membantu peserta belajar topik web dasar dengan contoh interaktif.

2. Fitur Utama

  • Halaman daftar kursus (filter, pencarian)
  • Halaman materi (teks, contoh kode, video)
  • Kuis singkat dengan penilaian otomatis
  • Mini project dengan penyimpanan lokal
  • Admin sederhana untuk menambah materi

3. Wireframe & UI Flow (lo-fi)

Homepage
  ├─ Header (logo, nav)
  ├─ Hero / CTA
  ├─ Daftar Kursus (grid)
  └─ Footer

Course Page
  ├─ Breadcrumb
  ├─ Judul & deskripsi
  ├─ Materi (tabs: Text, Video, Kuis)
  └─ Tugas / Download

4. Pembagian Tugas (jika kelompok)

  • Frontend: layout, styling, responsive
  • Backend (opsional): API sederhana atau mock data
  • UX/UI: wireframe, assets, ikon
  • QA: testing, bug report, aksesibilitas

5. Catatan Teknis & Tools

  • HTML, CSS (Flex/Grid), JavaScript
  • localStorage untuk menyimpan progress/hasil kuis
  • GitHub + Vercel untuk deploy
