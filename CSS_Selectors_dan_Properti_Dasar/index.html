Belajar CSS Selector & Properti Dasar

CSS digunakan untuk memperindah tampilan HTML. Berikut contoh penggunaan selector dan properti dasar sekaligus meperlihatkan tampilan nyata dari penggunaan css di halaman ini untuk memperindah tampilan:

Selector Tag

Selector tag akan mempengaruhi semua elemen dengan nama tag tertentu, misal semua <p>.

Selector Class

Selector class menggunakan tanda titik (.), misal .highlight untuk memberi warna khusus pada paragraf ini.

Selector ID

Selector ID menggunakan tanda pagar (#), misal #khusus untuk memberi style unik pada paragraf ini.

Properti Dasar CSS

  • color: mengatur warna teks
  • font-family: mengatur jenis font
  • margin: jarak luar elemen
  • padding: jarak dalam elemen
